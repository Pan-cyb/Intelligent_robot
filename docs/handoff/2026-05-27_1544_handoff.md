# 2026-05-27 15:44 Handoff

## Current Goal

新增药盒功能包：ROSA 说出对应药物后，通过 ROS 2 服务让药盒舵机转到对应药格位置。每个药物槽位间隔 90 度。

## Important Context

- 长期规则见 `AGENTS.md`。
- 当前阶段状态见 `docs/PROJECT_STATUS.md`。
- 已创建并切换到分支：`feature/medicine-box-servo`。
- RDK PWM 参考实现使用 `Hobot.GPIO.PWM(pin, frequency)`；运行前需要在 RDK 上启用对应 PWM 组，并确认 40pin 舵机信号脚。
- 药盒功能不进入 `task_manager` 状态机；它是独立硬件服务节点，ROSA 或其它节点通过 `/medicine_box/dispense` 调用。
