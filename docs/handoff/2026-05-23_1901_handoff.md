## Completed This Session

- `follower_nav2_controller.py` 已从整机状态机降级为纯跟随执行器：
  - 只订阅 `/robot_mode`、`/person_position`、`/person_distance`。
  - 只有 `/robot_mode == FOLLOWING` 时发送 Nav2 跟随 goal。
  - `/robot_mode != FOLLOWING` 时取消当前 goal 并空闲。
  - 移除 `IDLE/FOLLOWING/APPROACH_HAND/MEDICINE_READY` 整机状态。
  - 不再订阅或处理 `/fall_detected`。
  - person lost 只取消跟随 goal 并记录日志。
- `task_manager_node.py` 扩展整机模式：
  - `FOLLOWING`
  - `INSPECTION`
  - `EMERGENCY`
- `/robot_server/start_task` 新增：
  - `task_type="follow"`：进入 FOLLOWING，发布 `/robot_mode=FOLLOWING`。
  - `task_type="inspection"`：进入 INSPECTION，依次导航巡检点并观察 `/person_position`。
- `task_manager` 新增 emergency 处理：
  - 订阅 `/fall_detected`。
  - `fall_confirm_frames` 连续 true 后确认异常。
  - 抢占当前任务，取消 task_manager 自己持有的 Nav2 goal。
  - 切换 `/robot_mode=EMERGENCY`，让 follower 停止。
  - 发布 `/tts_text`：“检测到异常，您是否需要帮助？”
  - 新增 `/robot_server/clear_emergency` 和 `clear_emergency` service。
- `robot_server.launch.py` 扩展为主线服务端总入口：
  - 默认启动 `base_controller/navigation.launch.py`、`task_manager_node`、`tts_node`。
  - 默认启动相机、`base_link -> camera_link` 静态 TF、`person_tracker_node.py`。
  - 默认启动 `follower_nav2_controller.py`。
  - 如需关闭视觉跟随，显式传 `enable_person_tracker:=false enable_follower_controller:=false debug_window:=false`。
  - 支持 `debug_window`、`fall_confirm_frames`、`observe_duration_sec`、`person_seen_timeout_sec`。
- ROSA tools 增加：

  - `start_inspection_task()`
