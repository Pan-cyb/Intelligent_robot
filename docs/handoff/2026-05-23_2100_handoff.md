## Completed This Session

- `task_manager_node.py` 已小范围修复 Nav2 action 异步回调风险：
  - 新增 `_nav_goal_generation` 和 `_active_nav_goal_generation`。
  - `_send_navigation_goal()` 每次发送 Nav2 goal 时递增 generation 并绑定到 response/result 回调。
  - `_on_goal_response()` 和 `_on_navigation_result()` 忽略 stale generation。
  - `_reset_task()` 将 `_active_nav_goal_generation` 置 0，让取消、EMERGENCY、失败后的旧回调失效。
  - `_watchdog()` 覆盖 `NAVIGATION` 和 `INSPECTION`，修复巡检导航卡住不超时的问题。
- `rosa_agent/rosa_agent/action_tools.py` 新增：

```text
get_weather_advice(location)
```

- 天气工具使用 Open-Meteo 免 key API：
  - 先调用 geocoding API 解析地点。
  - 再调用 forecast API 获取未来小时预报。
  - 输出中文天气摘要和建议，例如收衣服、关窗、减少外出、补水、加衣。
- 修复 ROSA CLI 未调用天气工具的问题：
  - `DEFAULT_TOOLS` 加入 `get_weather_advice`、`start_following_task`、`start_inspection_task`。
  - `rosa_cli` 直连路由识别“天气/气温/温度/下雨/降雨/雨”等关键词。
  - “今天成都天气怎么样”现在会直接调用 `get_weather_advice("成都")`。
- 更新 `docs/PROJECT_STATUS.md` 记录天气工具和 Nav2 action generation 修复。
